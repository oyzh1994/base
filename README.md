# 项目
###### 项目说明
这是一个使用java的集成库，支持i18n能力、ssh能力、事件总线、存储能力，以及各类其他例如字符串处理的实现，减少第三方库的依赖

###### 依赖说明
1. jdk版本要求24

###### 结构说明 
common -> 基础模块，各种功能相关实现  
event -> 事件模块，事件总线实现  
i18n -> i18n模块，i18n资源及实现  
ssh -> ssh模块，ssh实现  
store -> 存储模块，存储实现

# Maven
###### 安装
mvn -X clean install -DskipTests

###### 注意
检查cmd里面java -version的版本号和项目版本号是否一致，否则可能出现无效的目标版本号24之类的问题
